OpenSter

Die Open-Source App fur Musik-Quiz-Enthusiasten. Scan QR-Codes, erstelle Quiz-Kartensets und teste dein Wissen uber Musikgeschichte.

GitHub Mehr erfahren

Features

📷

QR-Code Scanner

Scanne Spotify QR-Codes direkt von Plattenhullen, um Songs sofort abzuspielen.

🎵

Spotify Integration

Volle Integration mit Spotify Web API fur nahtlose Wiedergabe und Playlist-Verwaltung.

🎴

Quiz-Kartensets

Erstelle PDF-Quizkarten aus Playlists fur das ultimative Musik-Ratespiel.

🎨

Theme-System

Dark & Light Mode mit anpassbaren Akzentfarben, gespeichert in AsyncStorage.

📱

Cross-Platform

React Native mit Expo - lauft auf iOS, Android und im Web.

🔒

Open Source

100% Open Source unter MIT-Lizenz. Trage bei oder forke das Projekt!

Loslegen

1

Repository klonen

git clone https://github.com/grunz/OpenSter.git
cd OpenSter
2

Abhangigkeiten installieren

npm install
3

Entwicklung starten

npm start

Erfordert einen Spotify Developer Account. Konfiguriere Client ID und Secret in den Einstellungen.
